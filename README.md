# Toy Host Calendar

A React and TypeScript dashboard for managing toy host bookings, prep tasks, and availability in one place.

## Features

- Monthly calendar with booked and hold indicators
- Upcoming visit list with status reminders
- Prep checklist and availability summary

## Getting started

1. Install dependencies:

   npm install

2. Start the dev server:

   npm run dev

3. Build for production:

   npm run build

4. Preview the production build:

   npm run preview
